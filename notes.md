# RT-7 / APN-1 Notes

## Indicator

Pinout:

| Pin |   Function           |
|:---:|:---------------------|
| A | Power switch out |
| B | Power in (switch common) |
| C | Range switch out (in series with power switch) |
| D | GND / Indicator coil - |
| E | Indicator coil + |


Coil resistance: 100R
Full-scale current: 8.2mA
