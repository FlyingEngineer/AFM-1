# RT-7 / APN-1 Notes

## Indicator

Pinout:

| Pin |   Function           |
|:---:|:---------------------|
| A | Power switch out |
| B | Power in (switch common) |
| C | Range switch out (in series with power switch) |
| D | GND / Indicator coil - |
| E | Indicator coil + |


Coil resistance: 100R
Full-scale current: 8.2mA

## Altitude Selector

Switched potentiometer.

Pinout:

| Pin |   Function           |
|:---:|:---------------------|
| A | Casing |
| B | Top of track |
| C | Top wiper |
| D | Bottom wiper |
| E | Bottom of track |


Total resistance: Approx 36K
